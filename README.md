# 🌐 Simple Browser (Mini Web Browser)

## 📌 Project Description

This is a simple web browser application built using Node.js, Express.js, HTML, and CSS.
It allows users to enter a URL and fetch the content of the webpage.

---

## 🚀 Features

* Enter any website URL
* Fetch and display webpage content
* Handles invalid URLs with error message
* Simple and clean user interface

---

## 🛠️ Technologies Used

* Node.js
* Express.js
* HTML
* CSS
* Axios

---

## 📁 Project Structure

simple-browser/

├── server.js

│   └── style.css

---

## ▶️ How to Run the Project

1. Install dependencies:
   npm install

2. Start the server:
   node server.js

3. Open browser and go to:
   http://localhost:3000

---

## ⚠️ Notes

* Make sure internet connection is active
* Do not enter invalid URLs
